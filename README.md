# JEE 2027 & KCET Preparation Tracker 🚀

A professional, dynamic, and comprehensive progress tracker built specifically for JEE (Joint Entrance Examination) and KCET (Karnataka Common Entrance Test) aspirants targeting the 2027 exam cycle. 

## 🌟 Features

- **Personalized Dashboard**: A visually stunning overview of your overall progress, streak count, and daily focus.
- **Subject-Wise Analytics**: Track individual chapters for Physics, Chemistry, and Mathematics with custom categorization (High Weightage, Smart Lifters, etc.).
- **Interactive Study Planner**: Includes a Pomodoro timer and a weekly scheduling system to keep your study routine on point.
- **Data Persistence**: Your progress is safely stored right in your browser! Never lose your chapter checkmarks or daily logs.
- **Secure "Login" UI**: Simulates a login environment so your tracker feels like a premium SaaS product.
- **Resource Hub**: Quick links to the best free online resources and standard mock test platforms.

## 🛠️ Technologies Used
- **HTML5**: Semantic and accessible markup.
- **CSS3 (Vanilla)**: Advanced styling featuring glassmorphism, fluid layouts (Flexbox & Grid), micro-animations, and dynamic variables.
- **JavaScript (Vanilla)**: Handles local storage, state management, complex DOM manipulation, and interactive features like the Pomodoro timer without any heavy libraries.

## 🚀 Getting Started

This is a front-end only application! There is no complex build step required.

1. **Clone the Repository**
   ```bash
   git clone https://github.com/your-username/jee-tracker.git
   ```
2. **Open the App**
   Simply open the `index.html` file in your preferred modern browser (Chrome, Edge, Firefox, Safari).
   ```bash
   # On Windows
   start index.html
   ```
